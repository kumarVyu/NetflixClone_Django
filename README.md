# Netflix Clone (Django)

A simple Netflix-style web application built with **Django**.  
This project demonstrates how to create a video streaming platform with user authentication, movie browsing, and responsive UI.

---

## Features
- User registration and login
- Browse movies and TV shows
- Responsive Netflix-like UI
- Django templates and static files setup
- Easy to extend with your own content

---

## Tech Stack
- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (default, can be swapped with PostgreSQL/MySQL)
